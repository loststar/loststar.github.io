hexo只在特定页面开启mathjax

2020-08-04
  • hexo

在 Hexo-NexT 主题中，可以选择是否在所有页面加载 mathjax。但是在切换为 indigo 主题后，只能选择全局加载。对 themes/indigo/layout/_partial/plugins/mathjax.ejs进行少许修改即可实现。

  1. 在第2行加入 <% if (page.mathjax){ %>
  2. 在第24行加入<% } %>
  3. 在需要使用mathjax的文章头部加入 mathjax: true 即可
